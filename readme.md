# GIF Gallery
**Study** - It's a gif search gallery :)

## Made with

**Static Server**

Using [simple-files-server](https://www.npmjs.com/package/simple-files-server) module from npm.

**Front-end**
- Vanilla JS (using web components)

## How to run

```bash
npm install
```

**Development**
```bash
npm run dev
```

**Production**
```bash
npm start
```
